Petate.org

Built and designed an informational website for the Latinx community in Lexington.

Live Link

Project Overview

Petate.org is the communication platform for the Latinx community of Lexington. It is created for the ICT research project to understand the technology needs for the Latinx community as they have technology barriers in accessing reliable information including COVID19 data. The website was also built as an opportunity for the members of the community to interact with each other in Spanish about the local events or health topics.

This website is coded from scratch without the support of a content management system. It is built on PHP infrastructure and designed with HTML/CSS/Bootstrap. The login authentication and discussion board were manually made with PHP. There is some jQuery to add the interactive features. The COVID19 live data table is implemented with JSON, specifically from CovidActNow.org.
